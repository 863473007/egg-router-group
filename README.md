## Install

```bash
$ npm i egg-router-group --save
```

## Usage

```js
// {app_root}/config/plugin.js
exports.routerGroup = {
  enable: true,
  package: 'egg-router-group',
};
```

## Configuration

```js
// {app_root}/config/config.default.js
exports.routerGroup = {
};
```

see [config/config.default.js](config/config.default.js) for more detail.
